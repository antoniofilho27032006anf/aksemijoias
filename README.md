# AKsemijoias 💎

E-commerce completo de semijoias desenvolvido com tecnologias modernas, painel administrativo, autenticação, carrinho persistente e pagamentos PIX integrados via Mercado Pago.

# ⚙️ Instalação

## 1️⃣ Clone o projeto

```bash
git clone https://github.com/SEU-USUARIO/aksemijoias.git
```

---

## 2️⃣ Entre na pasta

```bash
cd aksemijoias
```

---

# 🔥 Backend

## Instalar dependências

```bash
cd backend
npm install
```

---

## Configurar .env

```env
DATABASE_URL="SUA_DATABASE_URL"

JWT_SECRET="SUA_JWT_SECRET"

CLOUDINARY_CLOUD_NAME="SEU_CLOUD_NAME"
CLOUDINARY_API_KEY="SUA_API_KEY"
CLOUDINARY_API_SECRET="SUA_API_SECRET"

MERCADO_PAGO_ACCESS_TOKEN="SEU_ACCESS_TOKEN"
```

---

## Prisma

```bash
npx prisma generate
npx prisma db push
```

---

## Rodar backend

```bash
npm run dev
```

Servidor:

```bash
http://localhost:3333
```

---

# 🎨 Frontend

## Instalar dependências

```bash
cd frontend
npm install
```

---

## Rodar frontend

```bash
npm run dev
```

Frontend:

```bash
http://localhost:3000
```
